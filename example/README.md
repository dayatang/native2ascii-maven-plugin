
# native2ascii-maven-plugin Example

## Run native2ascii-maven-plugin goal

```bash
mvn package
```

or run mvn `native2ascii-utf8-properties` exection id of goal `native2ascii`

```bash
mvn  native2ascii:native2ascii@native2ascii-utf8-properties
```
